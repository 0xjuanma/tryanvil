Anvil

A minimalist app to install and manage tools in batches using Homebrew. Streamline your developer tool setup with an intuitive interface.

Get Started

Download the latest version of Anvil for macOS. Installation is simple and takes just a few minutes.

Download for macOS Try Anvil CLI

Requires macOS 11.0 (Big Sur) or later

Features

•

Create Tool Groups

Create groups of tools to install and manage them in batches. Easily add and remove tools from groups to ensure you have the tools you need when you need them.

•

One-Click Install

Install entire tool groups with a single click. No need to install tools one by one.

•

Export/Import Tool Groups

Save and share your tool groups with export/import functionality. Easily import tool groups from other users or share your own tool groups with the community.
